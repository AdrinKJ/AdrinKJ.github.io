Wayfarer.

Wayfarer, our seamless travel booking website, combines HTML, JavaScript, CSS, PHP, SQL, Ajax, and Bootstrap to create an attractive, user-friendly interface. With stunning destinations, personalized experiences, and sustainability in mind, Wayfarer offers a responsive design for easy browsing. Its unique safety feature includes an emergency button that transmits user coordinates for prompt assistance, enhancing overall travel security.

Wayfarer Features:

1. User Authentication and Profiles
2. Tour Package Creation
3. Booking Management
4. Payment Integration
5. Feedback System
6. SOS Button For Tourists
7. Integration with External APIs

Spaceship Game

Retro Spaceship is a simple game built using pygame. This code helped me learn the basics of python and OOPs concepts. Its a multiplayer game, where 2 players fight to win the game. Each player can shoot upto 4 bullets at a time. If the bullet hits the other player then, a sound would be played, furthermore the opponent loses a point. The first player to beat the other player wins.
